# IMC-DADOS
# Calculadora de IMC

Projeto simples para cálculo do **Índice de Massa Corporal (IMC)**, desenvolvido em **Python puro** e **HTML/CSS/JavaScript**.  
O objetivo é demonstrar a separação entre **backend** e **frontend**, sem integração direta entre eles.
